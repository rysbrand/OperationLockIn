# 🔒 Operation Lock-In

Personal fitness + nutrition tracker. Built with Vite + React, hosted on GitHub Pages.

**Live:** https://rysbrand.github.io/OperationLockIn/

## Features
- 5-day workout program (heavy lifting + LISS + flexibility)
- 4-week meal rotation with daily macro totals
- Macro dashboard with micronutrient breakdown
- Interactive Costco grocery checklist

## Local Dev

```bash
npm install
npm run dev
```

## Deploy

Push to `main` — GitHub Actions builds and deploys automatically.

> Make sure GitHub Pages is set to **GitHub Actions** as the source in repo Settings → Pages.
